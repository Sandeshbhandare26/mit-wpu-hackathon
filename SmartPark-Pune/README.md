# SmartPark Pune

AI-Powered Smart Parking & Urban Mobility Management System for Pune City.

## Features
- Real-time Parking Spot Hunting
- AI-Driven Predictive Availability
- Seamless Payments & Bookings
- Smart Gate Access with Vehicle Verification
- Owner Dashboard for Revenue & Management
- Urban Flow Intelligence for City Planning

## Tech Stack
- **Frontend:** Vite + React + Tailwind + Framer Motion
- **Backend:** Firebase (Functions, Firestore, Auth)
- **ML:** Python + Flask + Scikit-Learn (Availability Prediction)
- **Maps:** Google Maps API / Leaflet
